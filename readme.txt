== Description ==
Hello Friends,<br>
Allows visitors of your site to share your site's content just by selecting the content that they wish to share.<br>

== Installation ==
	
* Upload the plugin files to the '/wp-content/plugins/' directory
* Activate the plugin through the 'Plugins' menu in WordPress
